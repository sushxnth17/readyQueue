About

ReadyQueue is a CPU scheduling visualizer built to help students and developers understand how different algorithms behave on the same workload.

It provides real-time execution timelines, performance metrics, and algorithm comparison to make learning more intuitive and practical.

Supported algorithms include FCFS, SJF, SRTF, Round Robin, and Preemptive Priority.
