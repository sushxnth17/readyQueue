# CPU Scheduling Simulator

A full-stack web application to visualize and understand CPU scheduling algorithms such as FCFS, SJF (Preemptive & Non-Preemptive), Round Robin, and Priority Scheduling.

---

## 🚀 Features
- Add and manage processes dynamically  
- Supports multiple scheduling algorithms  
- Visual Gantt chart representation  
- Calculates:
  - Waiting Time  
  - Turnaround Time  
- Clean and interactive UI
